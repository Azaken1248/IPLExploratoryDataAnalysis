# IPL Exploratory Data Analysis and Pipeline

## Overview
This repository contains a comprehensive data engineering pipeline and exploratory data analysis (EDA) framework for Indian Premier League (IPL) cricket data. The project is designed to ingest raw ball-by-ball and match-level datasets, process them through a structured pipeline, and extract actionable insights regarding player performance, team strategies, and match outcomes.

A key component of this analysis includes hypothesis testing on T20 batting strategies, specifically evaluating the statistical impact of "anchor" innings on a team's final score and win probability using Multiple Linear Regression.

## Repository Structure

**Notebooks**
* `IPLDataPipeline.ipynb`: The core data engineering workflow. Handles data ingestion, missing value imputation, type validation, and multi-source relational merging.
* `IPLCaseStudy.ipynb`: Focused analytical queries generating specific metrics such as strike rates, economy rates, boundary percentages, and phase-specific performance (e.g., powerplay vs. death overs).
* `IPLTrendAnalysis.ipynb`: Advanced statistical modeling and visualization, including the linear regression models used to test batting hypotheses.

**Directories**
* `dataset/`: Contains the raw input data.
  * `deliveries.csv`: Ball-by-ball granular match data.
  * `matches.csv`: Match-level metadata and outcomes.
* `output/`: Contains the structured CSV reports generated by the pipeline.
  * `anchor_analysis.csv`: Engineered features for regression modeling.
  * `runs_per_match.csv`, `team_scores.csv`, `top_batters.csv`, `strike_rate.csv`, `economy.csv`, `death_overs.csv`: Aggregated analytical deliverables.

## Technical Stack
* **Language:** Python 3.x
* **Data Manipulation:** Pandas, NumPy
* **Data Visualization:** Matplotlib
* **Machine Learning/Stats:** Scikit-learn

## Setup and Installation

1. Clone this repository to your local machine.
2. Ensure you have the required dependencies installed. You can install them using the following command:

```bash
pip install pandas numpy matplotlib scikit-learn
```

3. Ensure the raw datasets (`deliveries.csv` and `matches.csv`) are correctly placed within the `dataset/` directory before executing the notebooks.
